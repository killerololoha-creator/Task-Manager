# Task Manager Console Application

## Автор
Лукин Иван

## Описание программы
Консольное приложение для управления задачами с использованием ООП, MVC, очередей и стеков.

### Возможности:
- ✅ Добавление, редактирование, удаление задач
- ✅ Срочные задачи с дедлайном (наследование)
- ✅ Фильтрация по статусу и приоритету
- ✅ Очередь задач по приоритету (High → Low)
- ✅ Отмена последнего действия (Stack Undo)
- ✅ JSON сериализация
- ✅ Валидация ввода
- ✅ Красивый консольный интерфейс с эмодзи
